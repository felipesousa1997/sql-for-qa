# Subqueries

Nesta pasta estão exercícios utilizando Subqueries com foco em investigações de dados para Quality Assurance.

## Exercício 01 - Clientes com mais de 35 pagamentos

### Objetivo

Identificar clientes que realizaram mais de 35 pagamentos utilizando uma subconsulta com IN.

### Conceitos praticados

* Subquery
* IN
* GROUP BY
* HAVING
* COUNT

### Aplicação em QA

Esse tipo de consulta pode ser utilizado para validar regras de negócio, como identificar clientes frequentes, elegíveis para programas de fidelidade ou campanhas promocionais.
