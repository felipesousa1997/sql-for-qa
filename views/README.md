# Views

Esta pasta reúne exercícios utilizando **VIEW** no MySQL com o banco de dados **Sakila**.

## Objetivo

Aprender a criar e utilizar Views para simplificar consultas SQL e facilitar a reutilização de consultas complexas.

## Conceitos praticados

* CREATE VIEW
* CREATE OR REPLACE VIEW
* JOIN
* ORDER BY

## Exercícios

### Exercício 01 - Vendas por Cliente

**Objetivo:** Criar uma View que reúna informações dos clientes e seus pagamentos.

**Conceitos utilizados:**

* CREATE OR REPLACE VIEW
* JOIN
* ORDER BY

**Aplicação em QA:**

Views podem ser utilizadas para simplificar consultas frequentes durante a validação de dados em relatórios, dashboards e APIs, evitando a necessidade de escrever o mesmo JOIN repetidas vezes.
