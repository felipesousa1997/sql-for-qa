# SQL for QA - Estudos com Sakila

Repositório criado para registrar minha evolução em SQL com foco em Quality Assurance (QA).

## Banco utilizado

* MySQL
* Sakila Database

## Conceitos estudados

* SELECT
* WHERE
* ORDER BY
* LIMIT
* OFFSET
* LIKE
* DISTINCT
* JOIN
* GROUP BY
* SUM
* COUNT
* Investigação de dados

## Objetivo

Desenvolver habilidades de análise e validação de dados para atuação como QA, utilizando cenários próximos ao ambiente profissional.

## Casos de estudo

Além dos exercícios básicos, o repositório contém investigações simulando situações reais de QA:

* Validação de rankings
* Conferência de relatórios
* Verificação de clientes sem registros
* Análise de inconsistências em dados
